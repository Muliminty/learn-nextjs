# Next.js 学习项目

> 一个系统性的 Next.js 全栈开发学习项目，从基础概念到实战应用，最终完成一个包含前后端和数据库的完整项目。

## 📖 项目简介

这是一个 Next.js 学习项目，旨在通过循序渐进的方式掌握 Next.js 的核心概念和全栈开发技能。项目采用 **App Router** 架构，涵盖路由系统、组件开发、数据获取、API 路由、数据库集成、身份验证等核心知识点。

## 🎯 学习目标

完成一个完整的全栈应用，包含：

- ✅ 前端页面（多个页面，良好的 UI/UX）
- ✅ 后端 API（完整的 CRUD 操作）
- ✅ 数据库（SQLite，多个表，表关系）
- ✅ 身份验证（注册、登录、权限控制）
- ✅ 数据验证和错误处理
- ✅ 良好的代码组织和结构

## 🛠️ 技术栈

- **框架**: Next.js 14+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **数据库**: SQLite
- **ORM**: Drizzle ORM 或 Prisma（可选）
- **身份验证**: JWT 或 NextAuth.js

## 🗺️ 学习路径

### 第一阶段：基础概念（1-2天）
- Next.js 简介与环境搭建
- App Router vs Pages Router
- 服务端组件 vs 客户端组件
- Layout 和 Page 的理解

### 第二阶段：路由系统（2-3天）
- 静态路由和动态路由
- 路由组和并行路由
- 导航和路由守卫

### 第三阶段：组件开发（2-3天）
- 组件创建和组织
- 服务端和客户端组件实践
- 样式管理

### 第四阶段：数据获取（2-3天）
- 服务端数据获取
- 数据缓存和重新验证
- 加载状态和错误处理

### 第五阶段：API 路由（2-3天）
- Route Handlers
- 请求和响应处理
- 中间件

### 第六阶段：数据库集成（3-4天）
- SQLite 数据库设置
- CRUD 操作
- ORM 使用
- 数据库迁移

### 第七阶段：身份验证和授权（2-3天）
- 用户注册和登录
- Session 管理
- 路由保护

### 第八阶段：UI/UX 优化（2-3天）
- 表单处理
- 状态管理
- 图片优化
- SEO 优化

### 第九阶段：实战项目开发（5-7天）
- 完整的项目开发实践

### 第十阶段：高级特性（可选，3-4天）
- Server Actions
- 文件上传
- 实时功能
- 国际化
- 测试

## 🚀 快速开始

### 前置要求

- Node.js 18+ 
- npm、yarn 或 pnpm

### 安装步骤

1. **克隆项目**（如果是从仓库克隆）
```bash
git clone <repository-url>
cd learn-nextjs
```

2. **安装依赖**
```bash
npm install
# 或
yarn install
# 或
pnpm install
```

3. **启动开发服务器**
```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

4. **打开浏览器**
访问 [http://localhost:3000](http://localhost:3000)

## 📚 学习建议

1. **循序渐进**：按照学习规划的顺序完成每个阶段，不要跳跃
2. **动手实践**：每学一个概念，立即写代码实践
3. **做笔记**：在 `!doc` 目录下记录学习心得和遇到的问题
4. **遇到问题**：查看官方文档、搜索解决方案、记录问题
5. **定期回顾**：完成一个阶段后，回顾并总结
